# xkcd 
 xkcd - tools for xkcd comics

## Requirements 
- Python 3

## Scripts Included
    src/quick_info.py
Grab's title and alt for specified xkcd comic.

    src/download.py
Download specified xkcd as `xkcd_[number].png` into current directory.

    src/check_latest.py
Check the number and title of the latest xkcd.
